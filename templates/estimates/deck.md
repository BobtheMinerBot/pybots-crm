## Pricing Calculator

**Basic Deck (pressure treated):**
- Ground level: $25-35/sq ft
- Elevated (4-8 ft): $40-55/sq ft
- Elevated (8+ ft): $55-75/sq ft

**Composite Deck:**
- Ground level: $40-55/sq ft
- Elevated (4-8 ft): $55-75/sq ft
- Elevated (8+ ft): $75-100/sq ft

**Add for:**
- Curves/angles: +15-25%
- Over water: +20-30%
- Limited access: +10-15%

## Scope Description Template

```
SCOPE OF WORK: Deck Construction

ISLA Builders will construct a new deck at [ADDRESS]:

1. PERMITS & PLANNING
   - Obtain building permit
   - Survey/stake deck location
   - Call for utility locates

2. DEMOLITION (if applicable)
   - Remove existing deck structure
   - Remove old footings as needed
   - Dispose of all materials

3. FOUNDATION
   - Excavate for footings per plan
   - Pour concrete footings with post brackets
   - Install posts and verify level/plumb

4. FRAMING
   - Install ledger board with proper flashing
   - Install beams and joists per code
   - Install blocking and hurricane hardware
   - Frame for stairs as specified

5. DECKING
   - Install [MATERIAL] decking with proper spacing
   - Install fascia boards
   - Cut and finish all edges

6. RAILINGS & STAIRS
   - Install code-compliant railings
   - Build stairs with proper rise/run
   - Install stair railings

7. FINISHING
   - Final inspection
   - Touch-up and cleaning
   - Owner walkthrough

DIMENSIONS: [X] ft x [X] ft = [X] sq ft
ELEVATION: [X] ft above grade
MATERIALS: [Specify decking/railing materials]

TIMELINE: [X] working days
```

## Florida Keys Considerations

- All hardware must be stainless steel or hot-dipped galvanized
- Wind load requirements (check local codes)
- Flood zone considerations
- Salt exposure - recommend composite or IPE
- Permit required for all new decks
