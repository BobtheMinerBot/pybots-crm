## Pricing Calculator

**By Damage Level:**

*Minor (surface only):* $40-60/sq ft
- Waterproofing failed
- Minor cracking
- No structural damage

*Moderate (rebar exposed):* $65-90/sq ft
- Spalling present
- Rebar visible but sound
- Localized areas

*Severe (structural):* $100-150/sq ft
- Deep concrete loss
- Corroded rebar
- Structural concern
- May need engineer

## Scope Description Template

```
SCOPE OF WORK: Balcony Repair & Waterproofing

ISLA Builders will repair and waterproof the balcony at [ADDRESS]:

1. PROTECTION & ACCESS
   - Install protection below work area
   - Set up scaffolding/access as needed
   - Establish safe work zone

2. CONCRETE REPAIR
   - Remove all loose and damaged concrete
   - Expose corroded reinforcement
   - Clean rebar to bright metal
   - Apply rust-inhibiting coating
   - Install supplemental rebar as needed
   - Apply repair mortar in lifts

3. WATERPROOFING
   - Prepare surface (grind/clean)
   - Repair/install drains
   - Apply waterproof membrane
   - Install traffic-bearing top coat
   - Flash at wall connections

4. RAILING WORK
   - Remove railings for repair access
   - Repair post anchorages
   - Reinstall railings
   - [Or: Install new code-compliant railings]

5. FINISHING
   - Repair ceiling below (if applicable)
   - Stucco/finish repairs at edges
   - Final coating/paint

6. FINAL
   - Water test
   - Final inspection
   - Owner walkthrough

BALCONY SIZE: [X] sq ft
SCOPE: [Minor/Moderate/Severe repair]
TIMELINE: [X] working days
WARRANTY: 5-year waterproofing, 2-year workmanship
```

## Inspection Checklist

- [ ] Bounce test (feel for soft spots)
- [ ] Crack mapping
- [ ] Rebar cover depth (if exposed)
- [ ] Drain condition and location
- [ ] Railing post condition
- [ ] Ceiling below (water stains?)
- [ ] Slope (does it drain?)
- [ ] Wall flashing condition

---

## Red Flags → Recommend Engineer

- Visible sagging
- Cracks through full depth
- Heavily corroded/missing rebar
- Previous repairs failed
- Movement under load
- Separation from building

---

## Notes

- Balcony failures are life-safety issues — don't underestimate
- Always check railing post anchorage
- Waterproofing is critical in Keys environment
- Consider engineering letter for condos/HOAs
